Sobre mim

Olá! Sou Bruna Carla, tenho 20 anos e moro em Nova Iguaçu, Rio de Janeiro. Minha jornada na tecnologia começou com minha paixão por jogos online, o que despertou minha curiosidade pelo desenvolvimento. Sou formada em Análise e Desenvolvimento de Sistemas e me especializei em Desenvolvimento Web, com foco em construir interfaces modernas, acessíveis e responsivas. Tenho conhecimentos sólidos em HTML, CSS, JavaScript e frameworks como React. Além disso, ampliei minha formação com cursos na Udemy e Alura, e participei de eventos como o Discover da Rocketseat, onde aprofundei meus estudos em tecnologias web e boas práticas de desenvolvimento.

Principais Habilidades

React

Experiência com React, incluindo Hooks (useState, useEffect), React Router para navegação e Redux para gerenciamento de estado global.

TypeScript

Adição de tipagem estática ao JavaScript, melhorando a robustez e a manutenção do código em projetos de maior escala.

Node.js

Desenvolvimento de back-end com Node.js, incluindo a criação de APIs RESTful e integração com bancos de dados.

JavaScript

JavaScript é uma linguagem de programação essencial para o desenvolvimento web. Ela permite adicionar interatividade, manipulação de elementos DOM e comunicação assíncrona com o servidor.

Python

Python é uma linguagem de programação de alto nível, conhecida pela sua simplicidade e legibilidade. É amplamente utilizada em diversas áreas, como desenvolvimento web, automação, análise de dados.

Sass

Sass é uma extensão do CSS que permite usar variáveis, aninhamento de regras, mixins e outras funcionalidades, tornando o código CSS mais poderoso e organizado.

Bootstrap

Bootstrap é uma das frameworks front-end mais populares, oferecendo componentes prontos, como botões, formulários e navegação, além de um sistema de grid responsivo.

Tailwind CSS

Tailwind CSS é uma framework utilitária de CSS que oferece classes pré-configuradas para construção de layouts e componentes de maneira rápida e customizável.
